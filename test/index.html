Icons, without the SVG mess.

Devkitty is a lightweight icon rendering system that keeps SVGs out of the DOM, loads only what you use, and stays fast at any scale.

Get Started View Docs

Why Devkitty?

Devkitty renders icons using canvas from a compact GGDK file, keeping markup clean and icons protected.

No SVG in the DOM

Icons are rendered via canvas, not injected SVG nodes.

On-Demand Loading

Icons are parsed only when needed.

Preload Control

Immediate, idle, hover, viewport, or manual loading.

Framework-Agnostic

Works anywhere HTML works.
